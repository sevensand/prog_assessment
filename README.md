# prog_assessment

REQUIREMENTS

PHP 7.1.8
Composer version 1.5.1
Laravel Framework 5.5.32
Appache

https://github.com/sevensand/prog_assessment.git

go to the file directory using cmd

php artisan migrate - for database migration
php artisan key:generate
php artisan serve
